13:00 14:30 Prepare new warnings
Guidelines:
  • Issue maximum of 3 events per warning. Prepare separate warning documents if there are more than 3 events.
  • Lead times. The following are recommended earliest time to issue a warning:
    • Advisories (Green): 10 days
    • Watch (Yellow): 3 days
    • Alerts (Orange): 3 days
    • Warning (Red): 24 hours
  • Latest time to issue a warning: Even if the event is on-going, a warning can be issued (e.g. < 24 hours).
 📄 Warning template 🗺️ QGIS project for maps ✕
13:00 14:30 Update active warnings
Criteria for updating an active warning:
  1. Overall warning colour changes (e.g. from yellow to orange). If going to/from green/yellow to/from orange/red, contact DCP via telephone.
  2. Overall hazard type ("What") changes
  3. Location of any event ("Where") changes
  4. Timing of any event ("When") changes
 📄 Existing warnings ✕
13:00 14:30 Remove expired warnings
  1. Where event has ended and valid time has expired: Check the current warning document. If only advisories and/or watches (green/yellow), the warning should not be re-issued.
  2. Where an alert or warning (orange/red) is no longer needed: Prepare “All Clear” template. Example: Tropical cyclone has changed path, no longer affecting Zimbabwe.
